## 一、执行摘要

**YoTB** 是全球首款智能网球 IMU 传感器，基于 M5Stack ATOM S3（ESP32-S3 + MPU6886 六轴 IMU），将一颗 24mm 的微型传感器变成私人击球教练。

### 核心竞争优势

| 维度 | 竞品现状 | YoTB 优势 |
|------|---------|----------|
| 安装位置 | 全部装在球拍上 | **装在球上** — 全球首款智能网球 |
| 实时性 | 蓝牙赛后同步 | **50Hz WiFi 实时推流** |
| 查看方式 | 需下载专用 App | **内置 Web 仪表盘**，任何浏览器即开即用 |
| 连接方式 | 依赖手机蓝牙 | **WiFi AP 模式**，无需路由器，4 设备同连 |
| 旋转精度 | 间接推算 | **直接测量球体旋转**（IMU 在球内） |
| 数据开放 | 多数锁定在 App 内 | **CSV 一键导出 + 开源固件** |
| 硬件依赖 | 绑定特定品牌球拍 | **与任何球拍兼容** |
| 市场供给 | 大部分竞品已停产 | **正在进入一个竞品清空的市场** |

### 项目当前完成度

| 众筹核心承诺 | 代码状态 | 完成度 |
|-------------|---------|-------|
| 6 轴 IMU 数据采集 | `imu_logger/` — 200Hz 采样已实现 | 100% |
| 旋转分类（6 种类型） | TOPSPIN/BACKSPIN/SIDE_R/SIDE_L/SLICE/FLAT/MIXED | 100% |
| RPM 实时追踪 | 四元数积分 + 角速度计算 | 100% |
| WiFi AP + WebSocket | 50Hz JSON 推流 | 100% |
| Web 仪表盘 | 3D 网球 + 曲线 + 时间轴 + 统计 + CSV 导出 | 100% |
| 击球检测 | 4g 阈值 + 200ms 防抖 + 100ms 峰值追踪 | 100% |
| 3D 网球可视化 | 72 点参数化球缝 + 四元数驱动 | 100% |

**结论：YoTB 在软件/固件层面已达到众筹可交付的产品状态，这在硬件众筹项目中极为罕见。主要待解决的是产品化工程：定制外壳、电池集成、耐冲击性、量产方案。**

### 2.2 主要竞品深度对比

**Babolat Play Pure Drive（智能网球拍）**
- 价格：$299-$399（含拍）
- 传感器内嵌拍柄底部，6 轴 IMU
- 缺点：必须购买整支球拍；实时性差（赛后同步）；仅限 Babolat 用户
- 状态：2023 年后已逐步弱化该产品线

**Sony Smart Tennis Sensor SSE-TN1**
- 价格：$199（传感器）
- 缺点：2018 年后已停产；仅兼容特定球拍；App 已停止更新
- 状态：**已退出市场**

**Zepp Tennis**
- 价格：$149.99
- 9 轴传感器（双加速度计 + 双陀螺仪 + 磁力计）
- 缺点：需绑定在拍柄末端，影响手感；公司已转型高尔夫
- 状态：**网球产品线已停止**

**SwingVision（纯 App 方案）**
- 价格：免费 / $9.99 月 / $99.99 年
- 纯计算机视觉，使用手机摄像头
- 缺点：需要固定手机位置；依赖光线；精度不如 IMU
- 状态：活跃，获 USTA 官方合作

### 2.3 价格带分析

```
$0-$50     ：无直接竞品（市场空白区）       ← YoTB 早鸟价切入点
$50-$99    ：Coollang/QLIPP 类低端传感器（多已停产）
$99-$149   ：Zepp / HEAD / 大部分运动传感器核心价格带
$149-$199  ：Sony Smart Tennis Sensor
$199-$399  ：Babolat Play（含球拍）
$99+/年    ：SwingVision / PlaySight（纯软件订阅）
```

**关键洞察：** YoTB 的 Super Early Bird $39 定位在完全无竞品的价格空白区，且当前时机极佳——Sony、Zepp、Babolat Play 等主要竞品纷纷退出或停产，市场存在明显供给真空。

### 3.2 重点人群策略

**休闲玩家（最大基数）：** 以"社交分享"驱动转化——训练数据截图分享到朋友圈/Instagram。WiFi 直连 + 浏览器即开即用的"零门槛"体验是杀手锏。

**竞技选手（最高 ARPU）：** `ShotEvent` 结构体记录每次击球的 peakRPM、peakG、旋转轴分量和 spinType，最多 50 拍时间轴，直接满足数据化训练需求。

**教练/学院（B2B 高客单）：** WebSocket 支持 4 台设备同时连接（广播模式），教练用平板查看数据同时学员手机也看到实时反馈。AP 直连无需学院提供 WiFi 基础设施。

## 四、产品功能规划

### 4.1 设备端智能算法（软件升级）

| 功能 | 描述 | 优先级 | 开发周期 |
|------|------|--------|----------|
| **发球速度估算** | 冲量-动量定理 `F*dt = m*v`，基于 peakG + 100ms 峰值追踪 | P0 | 2-3 周 |
| **正手/反手/发球分类** | 冲击前后 200ms 窗口波形特征 + TFLite Micro 决策树 | P1 | 3-4 周 |
| **旋转一致性评分** | 同类型击球 RPM/旋转轴统计分析，映射为 0-100 分 | P0 | 1-2 周 |
| **疲劳检测** | 监测击球间隔变长、RPM 下降、G 力递减趋势 | P1 | 2 周 |
| **甜区命中检测** | 分析冲击后 50ms 频域特征（甜区单峰、非甜区二次振荡） | P2 | 3-4 周 |
| **Madgwick AHRS 滤波器** | 融合加速度计校正四元数漂移 | P0 | 1-2 周 |

### 4.2 训练模式与游戏化

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **目标挑战模式** | "连续 5 球 RPM > 120"、"旋转类型交替切换" | P1 |
| **实时评分系统** | 每次击球 A/B/C/D/F 评级 | P1 |
| **声音反馈** | 击球时浏览器播放不同音效，Web Audio API 实现 | P1 |
| **语音播报** | TTS 播报 RPM 和评级，学员无需看屏幕 | P1 |
| **连击系统** | 连续达标击球累加 combo 倍率 | P2 |

## 九、风险评估与缓解

### 9.1 核心风险矩阵

| 风险 | 严重程度 | 概率 | 缓解措施 |
|------|---------|------|---------|
| **球体耐久性** | 极高 | 高 | 硅胶灌封 + TPU 减震层 + 固态聚合物电池 + 定位为"训练用球" |
| **电池续航** | 高 | 高 | 去掉 LCD 省 30mA + 自适应推流频率 + 300-500mAh 电池目标 1.5-2h |
| **FCC/CE 认证** | 高 | 中 | 使用预认证 ESP32-S3-WROOM-1 模块简化流程，预算 $25K |
| **规模化制造** | 中 | 中 | 分阶段：首批基于开发板 + 3D 打印 → 自设计 PCB + 注塑 |
| **大品牌竞争** | 中高 | 低 | 球端传感新品类 + 零 App 体验 + 开源社区 + 快速迭代 |
| **交付延迟** | 中 | 高 | 承诺时间加 1 个月缓冲，每两周发更新 |

### 9.2 球体耐久性（最大技术风险）

**挑战：** 网球在比赛中承受 20-40g 冲击力。电子元件在反复高 G 冲击下可能脱焊、断裂。

**详细缓解策略：**
1. **灌封工艺** — 硅胶/环氧树脂完全灌封 PCB，消除焊点疲劳
2. **弹性安装** — TPU/硅胶减震层衰减冲击
3. **电池保护** — 固态聚合物电池 + 物理保护壳
4. **产品定位** — 明确为"训练用传感球"（训练场景冲击力 5-8g，远低于比赛）
5. **可更换外壳** — 外壳磨损可单独更换
6. **测试标准** — 1.5 米跌落 x 1000 次、10g 冲击 x 10,000 次

**建议：** 在 Kickstarter 之前完成至少 500 小时实际击球测试并公布数据。

### 9.3 核心护城河

1. **球端传感** — 所有竞品都是球拍端，YoTB 是第一个球端产品，全新品类
2. **零 App 体验** — WiFi AP + 浏览器仪表盘，大品牌习惯强制用户下载 App 难以复制
3. **开源社区** — MIT 许可 + 开放 WebSocket 协议，构建开发者生态
4. **速度优势** — 精简架构（4 个 main.cpp + 1 个 webpage.h），快速迭代
